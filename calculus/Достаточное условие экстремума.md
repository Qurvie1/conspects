Пусть f непрерывна в дельта окрестности x0 и дифференцируема в проколотой дельта окрестности x0. 
Тогда если производная меняет знак при переходе через точку x0, то x0 - точка локального экстремума
[[Теорема Ферма]]
[[Критерий дифференцируемости]]

По теореме Лагранжа о среднем:
f(x) - f(x0) = f'(кси(x))(x - x0)
Если f'(x) >= 0 в (x0 - дельта, x0) => f(x) <= f(x0) для любого x из этой полуокр
и f'(x) <= 0 в (x0, x0 + дельта) => f(x) >= f(x0) для любого x из x0, x0 дельта => x0 нестрогий локальный минимум
Теперь для высших производных:
Пусть существует n-ая производная в x0 (n >= 2)
При этом i-ые производные (x0) = 0 для всех i от 1 до n - 1, но f^(n) (x0) не равно 0.
Тогда если n - нечётное, то x0 - не явл точкой экстремума
Если n - чётно, и f^(n)(x0) > 0 - то минимум, если <0 - максимум
Так как существует n-ая производная, то можем разложить по Тейлору с остаточным членом в форме Пеано получим f(x) = f(x0) + сумма по всем от 1 до n - 1 (а это всё 0) + f(n)(x0) + o((x-x0)^n)
Тогда поделим на (x-x0)^n
f(x) - f(x0)/(x - x0)^n = f(n)(x0)/n! + e(x)
Если n - чётно, f^(n) > 0, тогда (x - x0)^n - положительно, а e(x) -> 0, тогда f(x) > f(x0) в некоторой окрестности x0, тогда x0 - строгий локальный минимум (тут так получается, что числитель положителен, а знаменатель отрицательный при x < x0, положительный в другом случае и получается, что у нас - меняется на + (то есть минимум))
Если n - нечётно, то знак числителя сохранится, но знак знаменателя меняется, то тогда знак числителя меняется => x0 не экстремум (ну какое-то из них стало больше или меньше)
